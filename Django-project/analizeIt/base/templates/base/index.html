AnalizeIt

AnalizeIt will take your Django project and show you important information about it. Using a new D3Js wrapper made by Daniel Carpio, it will display it using the recommendations according to 'Show me the number', a very important book written by Stephen Few

GitHub

Using GitHub API, AnalizeIt will take statistics of your project. It is very important your project is public, or else we will not be able to analize it.

Data-Driven Documents

D3 is a powerful JavaScript library to create all kinds of charts. We will use it display you the data, considering the recommendations made by Stephen Few in his book 'Show me the numbers'
